# Лабораторна робота №3
### Візуалізація даних (Data Visualization)

---

**Датасет:** [Automobile Data Set (UCI Machine Learning Repository)](https://archive.ics.uci.edu/ml/datasets/automobile)

---

## Опис проєкту
В рамках даної лабораторної роботи проведено завантаження, очищення (Data Cleaning) та глибокий візуальний аналіз набору даних **Automobile**. 

Метою роботи було застосування просунутих технік візуалізації для багатовимірних даних (Multidimensional Data). Було побудовано **6 професійних графіків** із детальним налаштуванням осей, легенд, статистичних маркерів та анотацій:

1. **Scatter Plot** із лінією регресії та анотаціями (залежність ціни від об'єму двигуна).
2. **Гістограма** розподілу потужності з кривою щільності (KDE) та блоком статистики.
3. **Багатовимірний Scatter Plot** (відображення 4 вимірів на одній площині).
4. **Комбінований графік (Boxplot + Swarmplot)** для детального аналізу розподілу цін.
5. **Маскована теплова карта (Masked Heatmap)** кореляції числових ознак.
6. **Стовпчаста діаграма (Bar Chart)** із виділенням ТОП-3 лідерів та підписами значень.

## Структура директорії
- `lab3.ipynb` — основний файл (Jupyter Notebook) з кодом, візуалізаціями та коментарями.
- `requirements.txt` — список необхідних Python-бібліотек для запуску.
- `.gitignore` — конфігураційний файл для виключення системного сміття з репозиторію.
- `README.md` — файл з описом проєкту та інструкціями.

## Вимоги до середовища
Для коректної роботи скриптів необхідний **Python 3.8+** та наступні бібліотеки:
- `pandas`
- `numpy`
- `matplotlib`
- `seaborn`
- `jupyter`

## Інструкція із запуску
1. Створіть та активуйте віртуальне середовище: python -m venv venv
2. Встановіть необхідні залежності: pip install -r requirements.txt
3. Запустіть Jupyter Notebook: jupyter notebook
4. Відкрийте файл з кодом лабораторної роботи та виконайте комірки по черзі.
