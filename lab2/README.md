# Лабораторна робота №2: Наука про дані

## Опис завдання
Метою лабораторної роботи є ознайомлення з процесом збору, очищення та початкового аналізу даних. Робота складається з двох частин:
1. **Частина 1:** Автоматизоване завантаження VHI-індексів для адміністративних одиниць України (дані NOAA) за допомогою `urllib`, очищення даних у `pandas dataframe`, зміна індексації областей на українську абетку та реалізація функцій для формування вибірок і статистичних розрахунків (екстремуми, медіана, середнє).
2. **Частина 2:** Завантаження та очищення великого датасету "Individual Household Electric Power Consumption". Формування складних вибірок (за потужністю, силою струму, часовими проміжками), профілювання часу виконання коду за допомогою модуля `timeit`. Нормування, стандартизація, розрахунок коефіцієнтів Пірсона та Спірмена, а також One Hot Encoding категоріальних ознак.

## Вимоги до системи
- **ОС:** Windows / macOS / Linux
- **Інтерпретатор:** Python 3.9+
- **Менеджер пакетів** `pip`.
- **Залежності:** Всі необхідні бібліотеки вказані у файлі `requirements.txt`.

## Інструкція із встановлення залежностей
Для ізоляції проєкту необхідно використовувати віртуальне середовище. Відкрийте термінал у папці `lab2` та виконайте наступні команди:

1. Створення віртуального середовища:
   python -m venv venv
2. Активація віртуального середовища:
   source venv/Scripts/activate (через Git Bash / PowerShell)
3. Встановлення необхідних бібліотек:
   pip install -r requirements.txt
4. Запуск Jupyter Notebook
   jupyter notebook
5. Виконання скриптів
   Відкрийте файли `part1.ipynb` і `part2.ipynb`, виконайте комірки.
