## Завдання на лабораторну роботу

Метою лабораторної роботи є створення програми з інтерактивним графічним інтерфейсом для візуалізації та обробки сигналів. 
Згідно з умовою, необхідно було виконати наступні кроки:

1. **Генерація сигналу:** Створити функцію гармоніки вигляду $y(t) = A\sin(\omega t + \phi)$ та накласти на неї гауссівський шум.
2. **Інтерактивний інтерфейс:** Розробити графічний інтерфейс користувача (GUI), який містить:
   * Поле для відображення графіків.
   * Слайдери для керування параметрами гармоніки (амплітуда, частота, фаза).
   * Слайдери для керування параметрами шуму (середнє значення, дисперсія/коваріація).
   * Чекбокс для увімкнення/вимкнення відображення шуму на графіку.
   * Кнопку "Reset" для скидання всіх налаштувань до початкового стану.
3. **Логіка оновлення:** Забезпечити миттєве оновлення графіка при зміні будь-якого параметра. Важлива умова: при зміні параметрів гармоніки структура шуму має залишатися незмінною, і навпаки.
4. **Фільтрація:** Відфільтрувати зашумлений сигнал за допомогою обраного цифрового фільтра та відобразити результат (відфільтровану гармоніку) поруч із початковими даними.

## Опис реалізованого функціоналу

Для виконання завдання було обрано мову програмування Python та бібліотеку `matplotlib` для побудови графіків і створення інтерактивних віджетів. 

Програма успішно реалізує всі поставлені вимоги. У додатку працює наступний функціонал:

* **Математична модель (Функція `harmonic_with_noise`):** Програма розраховує координати для трьох графіків одночасно: чистої ідеальної гармоніки, зашумленого сигналу та відфільтрованого результату.
* **Стабільність шуму:** Реалізовано ключову вимогу щодо незалежності шуму. Базовий масив випадкових чисел (шумовий "візерунок") генерується лише один раз при старті програми. Завдяки цьому пересування повзунків амплітуди чи частоти змінює лише форму самої хвилі, а шум масштабується та зміщується коректно, не генеруючись наново.
* **Система цифрової фільтрації (Функція `apply_filter`):** Для очищення зашумленого сигналу застосовано IIR-фільтр Баттерворта низьких частот (модуль scipy.signal). У графічний інтерфейс додано додатковий слайдер **"Filter Cutoff"**, який дозволяє користувачеві в реальному часі налаштовувати частоту зрізу фільтра, підбираючи оптимальне значення для максимального наближення відфільтрованої лінії до оригінальної синусоїди.
* **Графічний інтерфейс користувача (GUI):**
  * Графік автоматично масштабує вісь Y (`autoscale`) при зміні розмаху коливань, щоб сигнал завжди залишався в межах видимості.
  * Наявні 6 слайдерів: `Amplitude`, `Frequency`, `Phase` (для гармоніки), `Noise Mean`, `Noise Covariance` (для завад) та `Filter Cutoff` (для фільтра).
  * Робочий чекбокс **"Show Noise"** миттєво приховує зашумлену лінію, залишаючи для аналізу лише чисту та відфільтровану хвилю.
  * Кнопка **"Reset"** коректно обнуляє всі змінені значення до дефолтних.
* **Візуальне розділення:** Для зручності аналізу графіки розділено за стилями: чиста гармоніка відображається суцільною червоною лінією, зашумлений сигнал — напівпрозорою помаранчевою, а результат роботи фільтра — пунктирною синьою лінією.
